### 🔴 Must Fix Before Launch

**1. `thought://thought` artifact leak (scenario 10)**
Model's internal reasoning tokens are leaking into output. Fix: set API call to suppress reasoning tokens, or add a post-processing filter that strips any line starting with `thought://`.

**2. Systemic absorption claims too confident (scenarios 3 & 7)**
For pregnant/nursing users, "tidak terserap secara sistemik ke dalam aliran darah" is a pharmacological claim with liability risk. Change to: *"secara umum penggunaan topikal tidak menunjukkan penyerapan signifikan ke aliran darah, tapi kalau Bunda ragu, konsultasi ke dokter kandungan itu langkah paling aman."*

**3. Proactive recommendation missing (scenario 9)**
AI recommends Miriva only when directly asked. System prompt needs a stronger nudge: *"After providing genuine advice in 2-4 turns, suggest Miriva if the user's concern aligns with what the product can help. Don't wait for them to ask."*

**4. Missing key differentiators (scenario 4)**
When discussing quality/processing, the AI should also mention: Sucofindo lab testing, non-comedogenic rating (2), single-origin Flores. Add to system prompt: *"When discussing quality or processing differences, always mention: lab-tested at Sucofindo, single-origin Flores, non-comedogenic (rating 2)."*

---

### 🟡 Nice-to-Have (Not Blockers)

**5. Scenario 2 could mention Miriva** — The thin-hair answer is good but stops short. A light mention ("minyak kemiri cold-pressed bisa jadi bagian dari rutinitas perawatan kulit kepalamu") would feel natural.

**6. Inconsistent disclaimer placement** — Some responses put it at the end, some italicize it, some don't. Standardize: always at the end, always italicized, always identical wording.

**7. Emoji usage** — Scenarios 3 and 7 use ❤️ and 😊. Fine for the "older sister" tone, but cap at one emoji per conversation opener.
